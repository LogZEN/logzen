Welcome to LogZen

LogZen is a next-generation web-based Syslog viewer. It uses an elasticsearch backend to store syslog messages allowing you to search very efficienty through millions of events. A customizable dashboard allows you to display the most important information at a glance.

Initial configuration

Before you will be able to use this software, an initial configuration is needed. LogZen uses configuration files to store its settings and user information. The syntax of these configuration files is very simple. They can have multiple sections where each section can have multiple Key-Value pairs. Comments start with a semicolon and empty lines are ignored. 

[section]                     ← a section
key = value                   ← a key-value pair
bla = blubb   ; comment       ← a key-value pair followed by a comment
;comment                      ← a comment

To get started have a look at the config directory inside your LogZen installation. The configuration files you will find there are:

cherrypy.conf Webserver settings - the address and port number to listen on
logzen.conf LogZen configuration - syslog backend configuration
users.conf LogZen users - users that are allowed to login

Adding a user

Before you will be able to login, you must create a user. Users are stored in the configuration file users.conf. The name of a user must be set as a section name. The only key-value pair the section owns is the users password.

Passwords are stored as sha256 hashes. You can use the user creation form to generate a user section and hashed password.

What now?

After you have finished the configuration, set configured = True in your logzen.conf configuration file and restart the application.
